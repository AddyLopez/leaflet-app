Leaflet Tutorial

Background Information:

According to its home page, Leaflet was created "11 years ago by Volodymyr Agafonkin, a Ukrainian citizen" and is "the leading open-source JavaScript library for mobile-friendly interactive maps." This web page provides a short step-by-step tutorial for developers to get started with Leaflet in order to build interactive map features that can be integrated with other data. After learning how to use Leaflet, I wanted to spread that knowledge to my development community and also pay a small tribute to the Ukrainian people by supporting Ukranian web development.

Steps:

  1. Import Leaflet's CSS and JS by copying and pasting the link and script tags from the documentation.
  2. Create a div and assign it an ID to target the map.
  3. Inside a style tag in the head, assign width and height to the map using the ID's selector (width: 100% and height: 100vh will render a full-screen map). (For more involved styling, consider using a separate stylesheet.)
  4. Add a script tag above the body's closing tag. (Consider using a separate JS file for more involved projects.)
  5. Inside the script tag, declare a map variable and use the syntax from Leaflet's overview. Explanation of the syntax below:
    • Here is the sample given:
      var map = L.map('map').setView([51.505, -0.09], 13);
      • On the left side of the equal sign, the variable named "map" is declared. (Note to new developers: "var" is the pre-ES6 convention for declaring variables. Substitution of "var" by either of the now-common keywords "let" or "const" is fine here.
      • The "L" signifies "Leaflet."
      • After the dot operator and the function "map," include the map's ID from step 2 as the argument. Wrap the ID in quotation marks.
      • The purpose of the setView method is to determine the default coordinates and zoom level for the map. The method takes the coordinates [latitude, longitude] as the first argument and an integer designating the zoom level as a second argument.
        • A note about the zoom level: decrement the integer to zoom out; increment the integer to zoom in.
  6. Include the next line of code from Leaflet's overview to add the tile layer to the map.
    • Here is the sample given:
      L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {attribution: '© OpenStreetMap contributors'}).addTo(map);
      • Note: Copy the attribution's code directly from the documentation, not from the sample given above. The browser's user agent rendered the symbol for copyright and the anchor link wrapping "OpenStreetMap," interpreting the code instead of faithfully transcribing it character by character here.
    • Assign this code to a variable (e.g. const openStreetMap = L.tileLayer etc.)
    • Call the addTo(map) method on this new variable (e.g. openStreetMap.addTo(map) for improved legibility).
